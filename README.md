# Multi-size ICO 產生器

純前端的 `.ico` 多尺寸封包工具，無需安裝任何套件，直接以瀏覽器開啟 `index.html` 即可使用。

## 功能

- 上傳一個現有的 `.ico` 檔案作為來源圖示
- 自動縮放並產生以下 9 種尺寸的 PNG 影像：
  `16 × 16`、`24 × 24`、`32 × 32`、`40 × 40`、`48 × 48`、`64 × 64`、`96 × 96`、`128 × 128`、`256 × 256`
- 依照 ICO 格式規範將所有尺寸封裝成單一 `Multi-size.ico` 檔案
- 自動觸發下載
- 頁面下方即時顯示各尺寸的縮圖預覽

## 使用方式

1. 以瀏覽器直接開啟 `index.html`（或架設靜態伺服器）
2. 點選「選擇檔案」，選取來源 `.ico` 圖示
3. 點選「產生 Multi-size.ico」按鈕
4. 瀏覽器會自動下載 `Multi-size.ico`

> **注意**：來源圖示尺寸建議至少為 256 × 256，以確保縮小後各尺寸品質。

## 專案結構

```
MultiSizeIcon/
├── index.html   # 使用者介面與樣式
└── app.js       # 圖片載入、Canvas 縮放、ICO 封裝與下載邏輯
```

## 技術說明

| 項目 | 說明 |
|------|------|
| 執行環境 | 純瀏覽器，無後端、無套件相依 |
| 圖片縮放 | HTML5 Canvas API（`imageSmoothingQuality: "high"`） |
| ICO 封裝 | 手動組裝 ICO 二進位格式（ICONDIR + ICONDIRENTRY × 9 + PNG 資料） |
| 下載 | `Blob` + 動態建立 `<a>` 元素觸發下載 |
| 語系 | 繁體中文（`lang="zh-Hant"`） |
